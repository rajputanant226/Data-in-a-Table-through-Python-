# Displaying Tabular Data in Tkinter using ttk.Treeview
# Overview

This project demonstrates how to display tabular data inside a Python GUI application using Tkinter’s ttk.Treeview widget.
Treeview is one of the most powerful widgets in Tkinter for creating tables, data grids, dashboards, and admin panels.

You will learn how to build a table, insert dynamic rows, add scrollbars, and style the UI for a professional look.

# Features

✔ Create a GUI window in Tkinter
✔ Add a table using ttk.Treeview
✔ Define columns & headers
✔ Insert data dynamically
✔ Add vertical & horizontal scrollbars
✔ Apply styling for clean UI
✔ Perfect for dashboards, CRUD apps, record management systems
